# UI 设计规范

## 1. 色彩系统

### 1.1 风险色标（强制统一）
| 名称 | 色值 | 用途 |
|------|------|------|
| 🔴 Critical | `#FF4444` | 高危告警、严重威胁 |
| 🟠 High | `#FF8800` | 中危告警、可疑行为 |
| 🟡 Medium | `#FFCC00` | 低危告警、需关注 |
| 🟢 Low | `#00CC66` | 正常流量、安全状态 |

### 1.2 底色系统（暗色科技风）
| 名称 | 色值 | 用途 |
|------|------|------|
| Base | `#0D1117` | 页面底色 |
| Elevated | `#161B22` | 卡片/面板/侧边栏 |
| Surface | `#1C2333` | 弹出层/下拉菜单 |
| Hover | `#21283D` | 鼠标悬停 |

### 1.3 文字色
| 名称 | 色值 | 用途 |
|------|------|------|
| Primary | `#E6EDF3` | 主要文字 |
| Secondary | `#8B949E` | 次要说明 |
| Muted | `#484F58` | 禁用/占位 |

### 1.4 点缀色
| 名称 | 色值 | 用途 |
|------|------|------|
| Blue | `#58A6FF` | 主按钮、链接 |
| Cyan | `#39D2C0` | 科技感装饰 |
| Purple | `#BC8CFF` | 次要强调 |

## 2. 布局规范

```
┌──────────────────────────────────────────┐
│ Sider │  TopFilter          │             │
│ 240px │  [筛选条件...]       │  Content    │
│       ├─────────────────────│  Area       │
│ Nav   │                     │             │
│ Menu  │   Page Content      │             │
│       │                     │             │
│       │                     │             │
└──────────────────────────────────────────┘
```

- 左侧导航宽：240px，可折叠至 80px
- 顶部筛选栏高：56px
- 内容区 padding：24px
- 卡片圆角：8px
- 边框色统一：`#30363D`

## 3. 组件规范

### 3.1 KPI 统计卡片
- 最小高度 120px
- 标签 13px secondary，数值 32px bold，趋势 12px muted
- 悬停时边框变 accent-blue

### 3.2 告警表格行
- 高危行：左侧 3px `#FF4444` 色条 + 浅红背景
- 行悬停：`#21283D`

### 3.3 按钮
- 主操作（检测/确认）：蓝底
- 危险操作（拉黑）：红字幽灵按钮
- 次要操作（导出/标记）：默认幽灵按钮

## 4. 图表配色
- 图表中表示风险等级时，使用标准风险色
- 攻击类型分类饼图：Mirai `#FF4444`, Gafgyt `#FF8800`, Hajime `#FFCC00`, 其他 `#58A6FF`
- 正常流量：`#00CC66`

## 5. 字体
- 系统默认无衬线字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "PingFang SC", "Microsoft YaHei", sans-serif`
- 代码/日志：`Consolas, "Courier New", monospace`
