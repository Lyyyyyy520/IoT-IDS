# 项目需求规格

> 基于轻量化深度学习的智慧社区 IoT 僵尸网络入侵检测系统
> 天津市大学生创新训练计划（校级）| 2026.05 – 2027.05

---

## 1. 项目简述

面向智慧社区 IoT 设备安全需求，构建 CNN+LSTM 混合深度学习模型，依托树莓派实现本地离线入侵检测。系统可识别 Mirai、Gafgyt 等典型僵尸网络攻击，配套专业 Web 可视化界面。

## 2. 核心功能

### 2.1 检测能力
- **数据集**：N-BaIoT + BoT-IoT，约 650 万有效样本
- **特征集**：46 维 → 21 维社区专属特征（三层筛选）
- **模型**：轻量化 CNN（深度可分离卷积）+ LSTM + 注意力机制
- **输出**：5 分类（正常 / Mirai / Gafgyt / Hajime / 其他攻击）
- **模式**：离线 PCAP 文件检测 + 实时网卡流量检测

### 2.2 UI 页面
| 页面 | 功能 |
|------|------|
| 驾驶舱仪表盘 | KPI 卡片、流量曲线、攻击分布、近告警列表 |
| 告警中心 | 高危置顶、多维筛选、重复合并、快捷处置 |
| 分析视图 | 网络拓扑、攻击热力、流量图表、MITRE 链路 |
| 系统配置 | 检测模式、模型管理、告警规则、数据导出 |

### 2.3 部署形态
- **Windows PC**：完整管理控制台（React 前端 + Flask 后端 + ONNX 推理）
- **树莓派 4B**：轻量边缘检测引擎（Scapy 抓包 + 特征提取 + ONNX 推理）

## 3. 非功能需求
- 深色科技风 UI，红/橙/黄/绿标准化风险色
- 左侧导航 + 顶部筛选 + 中央内容布局
- 首次启动预置 Demo 数据，界面不空
- 单用户本地运行，无需登录/权限系统
